ヤクピタ

帳票監査、調剤確認、充填確認をカメラ読み取りでつなぎ、現場の「これで合っている」をすばやく支えます。

アプリをダウンロード できることを見る ヤクピタくんを見る
帳票監査 調剤確認 充填確認 GS1・バーコード読み取り

薬局の確認作業を、ひとつの画面から。

ヤクピタは、帳票の文字起こしから薬品照合、充填時の読み取り確認までを、現場で迷わず使える流れにまとめます。

監

帳票監査

帳票を撮影し、文字情報を読み取って薬品リスト化。監査の入口をすばやく整えます。

調

調剤確認

処方内容と読み取り結果を照合し、確認の抜けや見落としを減らします。

充

充填確認

薬品と充填先のコードを読み取り、取り違えを起こしやすい作業を支えます。

読み取って、照合して、記録する。

日々の確認手順に合わせて、必要なモードを選び、カメラで読み取り、結果を確認します。

モード選択 帳票監査、調剤、充填から作業を選択。
カメラ読み取り 帳票、GS1、バーコードをその場で読み取り。
照合結果確認 薬品名やコードの一致を画面で確認。
作業完了 確認済みの結果を残し、次の作業へ。

Android 端末へ、すぐに配布。

APK ファイルを端末で開くと、ヤクピタをインストールできます。薬局内での検証用に、最新版の配布ファイルをここから取得できます。

APK をダウンロード QRだけ表示 使い方を見る

URL / QR コード
https://raw.githubusercontent.com/fyuuchan-lgtm/GX_handy_GE_VER2/master/docs/landing/yakupita/downloads/yakupita-latest.apk

LINE などで上の URL を送ると、受け取った端末から APK を直接ダウンロードできます。

ファイル名: yakupita-latest.apk
対応端末: Android 9 以降

インストール時に「提供元不明のアプリ」を許可する必要があります。公開配布では署名済み release APK を配置してください。
